# DataVizTok

A TikTok-style interface for exploring random Data Visualizations from multiple platforms.

## Features

- Vertical scrolling feed of data visualization dashboards
- Dashboards sourced from Tableau Public, Power BI, and NovyPro
- Dashboard previews with thumbnails, titles and descriptions
- Category filtering (Finance, Sports, Health, Education, Marketing and more)
- Like and share dashboards directly or copy links
- Preloading of images and content for smooth scrolling
- Responsive design that works on mobile and desktop
- Progressive Web App (PWA) support for installing as a standalone app

## Tech Stack

- React 18
- TypeScript
- Tailwind CSS
- Vite

## Development

Run the commands below in the `/frontend` folder.

Install dependencies:

```bash
bun install
```

Run development server:

```bash
bun run dev
```

No backend required!
